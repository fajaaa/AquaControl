# AquaControl
AquaControl je desktop aplikacija namijenjena za evidenciju potrošnje i naplate vode. Omogućava jednostavno vođenje baze korisnika, unos mjesečne potrošnje, automatski obračun računa po tarifi, te praćenje uplata i dugovanja. Aplikacija generiše izvještaje o prihodima i dugovanjima, olakšava administraciju i doprinosi transparentnom upravljanju lokalnim resursima.
